ng-click

It gets triggered when the mouse clicks object

ng-change

It gets triggered when the dropdown or input field is changed. It also works on chackboxes and radio buttons

ng-focus

It gets triggered when any element get focused using tab key or click

ng-blur

It gets triggered when any element loses focus

ng-copy

It gets triggered when the user tries copy the text using right click or CTRL + C

ng-cut

It gets triggered when the user tries cut the text using right click or CTRL + X

ng-past

It gets triggered when the user tries paste the text using right click or CTRL + V

ng-dblclick

It gets triggered when the user double click the left button of the mouse on the object

keyborad Events ng-keydown, ng-keypress, ng-keyup

Keyborad events are basically triggered when the keys are pressed or released

  • ng-keydown

    is triggered when the keys is pressed it is on the down position

  • ng-keypress

    is triggered when the key is pressed. It does not recognise CTRL, ALT nor SHIFT

  • ng-keyup

    is triggered when the key is release
